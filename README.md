# Mockup Editor de Video

Este proyecto es un mockup de la interfaz de un editor de video desarrollado con HTML y CSS.

El objetivo del proyecto es representar visualmente la estructura de un editor de video moderno, incluyendo sus principales componentes de interfaz.

## Componentes incluidos

- Área de preview del video
- Timeline de edición
- Panel de recursos gráficos
- Panel de propiedades
- Barra de herramientas principal

## Tecnologías utilizadas

- HTML5
- CSS3

## Objetivo del proyecto

Este repositorio se utiliza para practicar el uso de Git y GitHub en un flujo de trabajo básico de desarrollo, incluyendo:

- Control de versiones
- Creación de commits
- Uso de ramas
- Pull Requests
- Resolución de conflictos
- Buenas prácticas de repositorio
